# 智能学习与科研助手平台 (SmartResearch) 需求分析文档

## 1. 项目愿景
SmartResearch 旨在打造一个超越传统文档管理的“多模态个人第二大脑（Second Brain）”。系统不仅能整理 PDF 和 Markdown 文本，更能深度解析图片、网页、视频及论文链接，通过 Agent 自动化理解与关联，构建统一的知识图谱，实现“知识点级”而非“文件级”的管理体验。

## 2. 核心功能需求

### 2.1 多模态知识来源支持
系统应支持以下异构知识源的接入与解析：

*   **文档知识**：PDF, Word, PPT, TXT, Markdown (如：Transformer 论文, 学习笔记)。
*   **图片知识**：JPG, PNG, JPEG (如：算法流程图, 实验截图, 课堂板书)。
*   **网页链接**：通用 URL, Arxiv 论文链接, GitHub 仓库链接, 公众号文章。
*   **视频链接**：B站、YouTube、MOOC 课程链接。
*   **第三方集成**：Notion 数据库同步、Obsidian 本地库监听。

### 2.2 自动化 Agent 处理流程

#### A. OCR & Vision Agent (图像理解)
*   **文字识别**：识别图片中的标题、正文及手写板书。
*   **公式识别**：解析数学公式并转化为 LaTeX 格式。
*   **语义理解**：识别图片类型（如“这是一个三层神经网络结构图”），提取结构化知识点。

#### B. Web Agent (链接解析)
*   **内容爬取**：自动抓取网页正文，过滤广告及无关干扰。
*   **摘要生成**：利用 LLM 提取长文摘要与核心关键词。
*   **论文解析**：针对 Arxiv 链接，自动获取摘要、作者、发布日期等元数据。

#### C. Video Agent (视频处理)
*   **语音转文字**：提取视频字幕或通过 ASR 技术生成文本。
*   **知识总结**：对视频内容进行分段总结，提取关键知识点。
*   **结构化输出**：生成课程知识树。

### 2.3 知识整理与图谱构建
*   **知识点解耦**：将不同来源（PDF, 图片, 网页）的相关内容统一抽象为“知识节点”，打破文件壁垒。
*   **自动关联**：Agent 根据语义自动发现新旧知识间的联系（如：新上传的图片与已有的 BERT 笔记相关）。
*   **主动推荐**：基于用户当前关注点，主动提示相关历史研究记录或相似文献。

## 3. 技术栈建议 (Python 后端)
*   **核心框架**：FastAPI / Django
*   **Agent 编排**：LangChain / LangGraph
*   **多模态处理**：
    *   OCR: PaddleOCR / GPT-4o Vision
    *   Web: Playwright / BeautifulSoup
    *   Video: Whisper (ASR)
*   **数据库**：
    *   向量库：ChromaDB / Pinecone (用于 RAG)
    *   图数据库：Neo4j (用于实体关系)

## 4. 项目亮点与创新
*   **全自动化流程**：用户上传 -> Agent 理解 -> Agent 关联 -> Agent 建图 -> Agent 推荐。
*   **多模态融合**：实现异构数据的统一语义表征。
*   **主动交互**：从“被动搜索”转向“主动知识推送”，真正作为科研和学习的智能伴侣。

## 5. 预期效果示例
用户上传：`Transformer 论文.pdf` + `结构图.png` + `Arxiv 链接`。
系统最终呈现：一个名为 **"Transformer"** 的知识主题，包含其核心原理（Attention, Encoder-Decoder）、关联技术（BERT, GPT, ViT）以及来自所有源的综合摘要。
