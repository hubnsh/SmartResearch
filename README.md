---

## 🚀 快速运行

**桌面版（推荐，下载即用）：**

```bash
cd SmartResearch
pip install -r requirements.txt
python desktop_app.py
```

**Web 版（浏览器访问）：**

```bash
cd SmartResearch
pip install -r requirements.txt
python run_server.py
# 打开 http://localhost:8002/js
```

**Windows 用户：** 双击 `SmartResearch/desktop_launcher.bat` 或 `SmartResearch/launcher.bat`。
